CentOS/RHEL 7
-------------

To add OpenNebula repository execute the following as root:

.. prompt:: bash $ auto

    # cat << EOT > /etc/yum.repos.d/opennebula.repo
    [opennebula]
    name=opennebula
    baseurl=https://downloads.opennebula.org/repo/5.7/CentOS/7/x86_64
    enabled=1
    gpgkey=https://downloads.opennebula.org/repo/repo.key
    gpgcheck=1
    #repo_gpgcheck=1
    EOT

Debian/Ubuntu
-------------

To add OpenNebula repository on Debian/Ubuntu execute as root:

.. prompt:: bash # auto

    # wget -q -O- https://downloads.opennebula.org/repo/repo.key | apt-key add -

**Debian 9**

.. prompt:: bash # auto

    # echo "deb https://downloads.opennebula.org/repo/5.7/Debian/9 stable opennebula" > /etc/apt/sources.list.d/opennebula.list
